# W3D_HW

HW1_XmasTree：

樹上的星星是開關可以控制燈泡，

每個燈泡都有固定5種顏色輪流變化。

初始是簡單模式，每個燈閃爍頻率相同，

按一下是複雜模式，每個燈閃爍頻率不同。

再按一下關燈。
